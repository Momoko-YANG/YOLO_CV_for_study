[Environment Setup]
(1) Create a new Python 3.10 environment using Anaconda:
conda create -n env_rec python=3.10
(2) Activate the created environment:
conda activate env_rec
(3) Install required dependencies using pip via requirements.txt:
pip install -r requirements.txt
